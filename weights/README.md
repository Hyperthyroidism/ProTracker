\# Model Weights



This directory is used to store the pretrained weights and trained checkpoints required by ProTracker.



\## Directory Structure



Please organize the weights as follows:



```text

weights/

├── README.md

├── yolo11/

│   └── yolo11\_vessel.pt

│

├── sam2/

│   └── sam2\_hiera\_large.pt

│

└── protracker/

&#x20;   └── protracker.pth

```



\## Weight Description



\### YOLO11 Detector



```text

weights/yolo11/yolo11\_vessel.pt

```



This file is the vessel detection model used to generate initial vessel candidates from UAV video frames.



The detector provides bounding boxes, confidence scores, and category labels for the following graph-based prompt optimization and SAM2 tracking modules.



\### SAM2 Model



```text

weights/sam2/sam2\_hiera\_large.pt

```



This file is the pretrained SAM2 checkpoint used for prompt-based video segmentation and mask propagation.



In ProTracker, SAM2 is guided by the optimized vessel prompts generated by the graph-based cascaded prompt optimizer.



\### ProTracker Checkpoint



```text

weights/protracker/protracker.pth

```



This file is the trained checkpoint of the graph-based cascaded prompt optimizer and related tracking components.



It is used to enhance node representations, refine edge features, and generate high-quality vessel prompts for SAM2-based tracking.



\## Usage



The weight paths are configured in the following files:



```text

configs/train/ProTracker.yaml

configs/test/ProTracker.yaml

```



For example, in the testing configuration:



```yaml

detector:

&#x20; model\_path: weights/yolo11/yolo11\_vessel.pt



graph\_prompt\_optimizer:

&#x20; checkpoint\_path: weights/protracker/protracker.pth



sam2:

&#x20; checkpoint\_path: weights/sam2/sam2\_hiera\_large.pt

```



\## Notes



The weight files are not included in this repository if their file sizes exceed the upload limit of GitHub.



Please manually place the required weight files into the corresponding folders before running training, testing, or demo inference.
